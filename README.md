# Instituto Esperança – Experiência Prática II

Site fictício para a ONG *Instituto Esperança*, desenvolvido como parte da **Entrega II – Estilização e Leiautes** da disciplina de CSS3. O projeto demonstra interface profissional, responsiva e acessível, utilizando **CSS Grid, Flexbox e sistema de design**.

---

## Funcionalidades

* **Menu principal** com dropdown e versão mobile (hambúrguer);
* **Cards de projetos** responsivos com hover e badges de categorização;
* **Formulário estilizado** com validação visual e feedback (alerts);
* Layouts **responsivos** para desktop, tablet e mobile (5 breakpoints);
* Sistema de design consistente: **paleta de cores, tipografia e espaçamentos modulados**.

---

## Tecnologias

* HTML5
* CSS3 (Grid, Flexbox, variáveis, breakpoints)
* JavaScript (menu mobile, máscaras de formulário e alerts)
